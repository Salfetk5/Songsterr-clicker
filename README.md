INSTALLATION AND USAGE GUIDE

Songsterr Clicker is a browser extension designed to ensure an uninterrupted music practice experience. It automatically clicks the "Continue" button whenever the site pauses playback, removes annoying ads, and allows you to customize hotkeys for seamless control.

KEY FEATURES:
- Auto-Click (250ms): Instant dismissal of synchronization windows and playback pauses.
- Cross-Platform: Compatible with both Desktop and Android mobile devices.
- Project File Tree: Manage your content efficiently with a custom folder system for one-click tab access.
- Interactive Control: Instant toggle functionality for the script via a dedicated button.

INSTRUCTIONS FOR DESKTOP (Windows / Mac / Linux):
1. Download and extract the extension archive into a folder of your choice.
2. Open your browser (Chrome, Edge, Opera, or Yandex) and navigate to the extensions page: chrome://extensions/
3. Enable "Developer mode" using the toggle in the top-right corner.
4. Click "Load unpacked" and select the folder where you extracted the files.
5. Go to songsterr.com and refresh the page (F5). The extension will activate automatically.

SECONDARY METHOD FOR DESKTOP (If the browser disables the extension):
1. Locate the .crx file in the folder.
2. Open your browser's extensions page.
3. Drag and drop the .crx file directly into the tab.
4. Click "Install extension."
5. Note: If you are using Yandex Browser and the icon is hidden, click the puzzle piece icon in the top-right corner, find the extension, and click the pin icon next to its name.

INSTRUCTIONS FOR ANDROID:
1. Install Kiwi Browser from the Google Play Store (this browser supports desktop extensions).

3. Enable "Developer mode."
4. Tap "+(from .zip/crx/user.js)" and select the downloaded archive from your phone's storage.
5. Open Songsterr in Kiwi Browser — it will work just like on a desktop.
