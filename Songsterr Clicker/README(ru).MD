ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ИСПОЛЬЗОВАНИЮ

Songsterr Clicker - это расширение для браузера, которое позволяет заниматься музыкой без прерываний. Оно автоматически нажимает кнопку «Продолжить», когда сайт приостанавливает воспроизведение, удаляет назойливую рекламу и позволяет настраивать горячие клавиши для управления процессом.

ОСНОВНЫЕ ФУНКЦИИ:
- Auto-Click 250мс: Мгновенное закрытие окна синхронизации и пауз.
- Cross-Platform: Поддержка ПК и мобильных устройств на базе Android.
- Project File Tree: Возможность создания папки с удобным открытием ссылки с табами по одному нажатию.
- Interactive Control: Мгновенное включение/выключение скрипта по кнопке

ИНСТРУКЦИЯ ДЛЯ ПК (Windows / Mac / Linux):
1. Скачайте и распакуйте архив с расширением в удобную для вас папку.
2. Откройте браузер (Chrome, Edge, Opera или Яндекс) и перейдите в раздел расширений по адресу: chrome://extensions/
3. Включите «Режим разработчика» (Developer mode) с помощью переключателя в верхнем углу окна.
4. Нажмите кнопку «Загрузить распакованное расширение» (Load unpacked) и выберите папку, в которую вы извлекли файлы.
5. Зайдите на сайт Songsterr и обновите страницу (клавиша F5). Расширение активируется автоматически.

ИНСТРУКЦИЯ ДЛЯ ПК 2(если браузер выключает расширение)
1.В папке найдите файл .crx
2.Откройте Ваш браузер
3.Зайдите в раздел расширений
4.Перетащите файл с расширением .crx в данную вкладку
5.Нажмите "Установить расширение"
(6).Если у Вас Яндекс, а так же если у вас расширение не показывается, Вам нужно нажать на пазл в правом верхнем углу, найти там расширение, и нажать на булавку справа от названия

ИНСТРУКЦИЯ ДЛЯ ANDROID:
1. Установите браузер Kiwi Browser из Google Play Store (этот браузер поддерживает расширения для ПК).
2. Нажмите на три точки в углу браузера и выберите раздел «Расширения».
3. Включите «Developer mode».
4. Нажмите кнопку «+(from .zip/crx/user.js)» и выберите архив с расширением в памяти вашего телефона.
5. Откройте Songsterr в Kiwi Browser — всё будет работать так же, как на компьютере.

3. ДЛЯ IPHONE (Safari) — Метод закладки
1. Скопируйте код из файла MOBILE_IPHONE.txt.
2. Создайте любую закладку в Safari, назовите её «Songsterr Fix».
3. Нажмите на иконку «Книги» (закладки) -> «Изменить».
4. Выберите свою закладку «Songsterr Fix».
5. Удали адрес сайта, вставь туда скопированный код и нажми «Готово».

КАК ПОЛЬЗОВАТЬСЯ НА IPHONE:
- Откройте песню на Songsterr.
- Нажмите на «Книги» (закладки) внизу экрана.
- Нажмите на созданную закладку «Songsterr Fix».
- Скрипт активируется мгновенно!
